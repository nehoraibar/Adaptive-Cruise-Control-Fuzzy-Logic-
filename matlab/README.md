# MATLAB Files

This folder contains the source files used to develop the Adaptive Cruise Control project.

## Files

### `simulink_model.slx`

Main Simulink model implementing the Adaptive Cruise Control system.

### `ACC_Fuzzy.fis`

Zero-Order Sugeno Fuzzy Inference System used by the controller.

## Requirements

- MATLAB R2024b (or compatible version)
- Simulink
- Fuzzy Logic Toolbox

Open `simulink_model.slx` in MATLAB and ensure `ACC_Fuzzy.fis` is located in the same directory or on the MATLAB path before running the simulation.
